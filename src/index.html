Football Goalkeeper Game

Welcome to the Football Goalkeeper Game! Before you start playing, please read the instructions below:

  • Wait for the game to start after positioning the field.
  • Swipe on the screen to shoot the ball.
  • Use the red dot as a directional arrow for shooting.
  • Adjust the speed of the shot using the given controller.
  • Your goal is to score points by shooting the ball into the goal.
  • Watch out for the goalkeeper and try to avoid being caught!
